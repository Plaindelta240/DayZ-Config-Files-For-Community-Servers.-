# DayZ-Config-Files-For-Community-Servers.-
These files contain boosted loot, full cars, boosted event, Gameplay changes, for Chernarus and Livonia. These files are great for new servers and can be adjusted at any time.
--------- 
These config files can be uploaded to your Nitrado server or to your mission folder. All these files are compatible with PC And Console Servers
CFGgameplay file -- this file has been modified for full stamina, standard Temp of 18c, player position shown on maps, build anywhere enabled, and terrain checkers disabled
Events file -- this file has increased Vehicles Spawns around the map, along with increased heli crashes. 
spawnableTypes file -- this file has full car parts with a jerrycan in the trunk of all vehicles, and gun parts have increased chances of spawning
